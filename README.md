# DhikrFlow 🌿

<p align="center"><img src="./assets/dhikrflow_logo.png" width=600px /></p>

DhikrFlow is a modern, minimal online Tasbih counter designed to support mindful dhikr.
It allows users to select zikr, view meanings, and track progress with a smooth and distraction-free experience.

---

## 🚀 Live Demo

👉 Visit Dhikrflow: https://dhikrflow.vercel.app

---

## ✨ Features

* 📿 Multiple dhikr selection (SubhanAllah, Alhamdulillah, etc.)
* 🌍 Meaning display for each zikr
* 🔢 Target-based counting system
* 🎯 Clean and minimal UI
* 📱 Fully responsive design
* ⚡ Lightweight and fast
